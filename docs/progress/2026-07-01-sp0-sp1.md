# SP0+SP1 Progress

Implemented:

- Next.js App Router scaffold with TypeScript, Tailwind, ESLint, Vitest, and Playwright scripts;
- self-hosted Supabase project configuration and environment template;
- foundation schema for profiles, pets, walker profiles, availability, storage, action logs, RLS, and generated TypeScript database shape;
- Supabase browser, server, and service-role clients using the Next.js 16 proxy/session pattern;
- e-mail/password signup, login, logout, and role selection for tutor and walker profiles;
- protected app shell, role-aware navigation, and dashboard entry points;
- tutor pet create, edit, list, and delete flow with validation and audit logs;
- walker profile, accepted pet sizes/behaviors, photo upload, active flag, and availability management;
- public walker browsing by region and route into walk-request creation;
- baseline architecture, data-model, endpoint, and README documentation.
